# 📢 Marketing Campaign Analysis

---

## 📌 Project Overview

This project analyzes marketing campaign data to evaluate campaign effectiveness, customer response, and ROI. The goal is to optimize future campaigns using data-driven insights.

---

## 🛠️ Tools Used

* 🐍 Python
* 📊 Pandas
* 📈 Matplotlib / Seaborn
* 📉 Excel
* 💾 CSV

---

## 📂 Dataset

* Marketing campaign dataset containing customer demographics, campaign details, and responses
* Features include:

  * `customer_id` 👤
  * `age` 👶👵
  * `gender` 🚻
  * `campaign_type` 📬
  * `channel` 📱💌
  * `response` ✅❌
  * `spend` 💰

---

## 🎯 Project Objective

* Measure campaign effectiveness
* Identify high-performing channels and customer segments
* Predict customer response to campaigns
* Provide recommendations for future campaigns

---

## 🧹 Data Cleaning Steps

* Removed duplicate entries
* Handled missing values ❓
* Encoded categorical variables 🔢
* Normalized numerical features 📏
* Split data into training and testing sets 🗂️

---

## 📊 Analysis Performed

* Campaign response rate by channel and type 📈
* Customer demographics vs response patterns 👥
* Impact of campaign spend on ROI 💸
* Visualized trends using plots and heatmaps 🖼️

---

## 🤖 Model Building

* Logistic Regression
* Random Forest 🌲
* XGBoost ⚡
* Evaluated using Accuracy ✅, Precision 🎯, Recall 📡, F1-score ⚖️, ROC-AUC 🖤

---

## 📈 Key Insights

* Email campaigns have higher response rates 📬
* Younger age groups respond better to social media campaigns 📱
* Higher spend does not always guarantee higher ROI 💸
* Random Forest performed best with X% accuracy 🏆
